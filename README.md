## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Customization

- Replace placeholder photos in `src/components/PhotoGallery.jsx` with your actual images
- Modify love messages in `src/components/LoveMessage.jsx`
- Adjust colors in `tailwind.config.js` if desired
